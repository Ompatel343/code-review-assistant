# Code Review Assistant

## Overview

Small full-stack app: upload source files, get LLM code review. Frontend is Vite + React + TypeScript + Tailwind. Backend is Express + SQLite.

## live demo

https://code-review-assistant-teal.vercel.app/

## Quick start

### Backend

```bash
cd backend
npm install
npm run dev
```
